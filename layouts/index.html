Why this page exists

A personal lab, not a polished brochure

我更关心一个工具在连续几周真实使用之后会不会开始漏数、变慢、漂移，还是只在 demo 里看起来很漂亮。 所以这里会同时放三种东西：正在做的工程、从真实 trace 长出来的公开信号、以及写下来的判断。

01
No fake evidence
截图、trace、和数据都应该来自真实运行，而不是为了展示临时捏出来的样子。
02
Infra should be observable
如果我每天都在用一套 AI 工具链，那它就应该能被追踪、汇总、对外解释。
03
APIs over vibes
当 Agent 开始影响真实决策时，估算和幻觉都不够，必须接入真实 API 和可复现脚本。
